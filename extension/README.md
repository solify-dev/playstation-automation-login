# PlayStation Auto Login Extension

## Install

1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select the `extension` folder.

## Usage

1. Open `https://www.playstation.com/` manually.
2. Click the extension icon.
3. Enter your sign-in email and password.
4. Click **Run Login**.

The extension will:

1. Click the landing page Sign In button.
2. Fill email.
3. Click Next.
4. Fill password.
5. Click Sign In.

## Notes

- Use only on accounts you own and are authorized to access.
- CAPTCHA/2FA may still appear and can block completion.
